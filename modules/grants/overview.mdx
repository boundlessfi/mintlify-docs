Grants on Boundless fund projects through an application process and milestone-based payouts. Funds are held in [trustless escrow](/concepts/trustless-escrow-explained) and released as milestones are completed and approved.

## Key concepts

- **Grant program** A funding pool and set of rules (e.g., focus area, eligibility). See [Grant programs](/modules/grants/grant-programs).
- **Grant types** Different structures (e.g., one-time, milestone-based). See [Grant types](/modules/grants/grant-types).
- **Application** Your proposal; see [Application process](/modules/grants/application-process).
- **Milestone delivery** How you submit work and receive payouts. See [Milestone delivery](/modules/grants/milestone-delivery).

## For applicants

- Find a [grant program](/modules/grants/grant-programs) that fits your project.
- Submit an application; if accepted, deliver [milestones](/modules/grants/milestone-delivery) to receive funding.

## For funders

- Create or sponsor grant programs and set criteria.
- Review applications and approve milestone completions to release funds.
