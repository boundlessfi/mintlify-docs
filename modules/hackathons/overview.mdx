## Hackathons on Boundless

Hackathons on Boundless are time-boxed events where participants build projects, compete for prizes, and can receive post-event milestone payouts. Funds are held in [trustless escrow](/concepts/trustless-escrow-explained).

## Key concepts

- **Participating** Joining as a builder or team. See [Participating](/modules/hackathons/participating).
- **Organizing** Running an event. See [Organizing](/modules/hackathons/organizing).
- **Judging system** How submissions are scored and winners chosen. See [Judging system](/modules/hackathons/judging-system).
- **Post-event milestones** Optional follow-up work and payouts after the hackathon. See [Post-event milestones](/modules/hackathons/post-event-milestones).
